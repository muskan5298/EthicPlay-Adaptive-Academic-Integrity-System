EthicPlay
Adaptive Serious Game for Academic Integrity and Ethical AI Usage

EthicPlay is a web-based educational serious game developed as part of a bachelor thesis at Riga Technical University (RTU). The project was designed to help students improve their understanding of academic integrity, citation practices, plagiarism prevention, and ethical AI-assisted academic writing.

The system combines scenario-based learning, gamification mechanisms, adaptive feedback, and citation analysis tasks to create an interactive learning environment focused on ethical decision-making during thesis development.

Project Objectives

The main objectives of EthicPlay are:

Improve student understanding of academic integrity
Increase awareness of proper citation practices
Teach responsible AI-assisted academic writing
Help students identify plagiarism and citation risks
Provide interactive ethical decision-making scenarios
Use gamification to improve engagement and learning motivation
Main Features
Scenario-based academic integrity challenges
Citation verification activities
Adaptive scoring and feedback system
Gamification mechanics (XP, levels, badges, streaks)
Integrity risk evaluation
Personalized learning progression
Final assessment challenge
Progress saving using browser local storage
Interactive educational dashboard

Adaptive scoring system

Final evaluation module
